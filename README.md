# Jogo de Adivinhação

Jogo simples em Java onde o jogador tenta adivinhar um número secreto entre 0 e 99 em até 5 tentativas.

## Como jogar

1. Execute o programa
2. Digite um número
3. Receba dicas se o número secreto é maior ou menor
4. Acerte antes de acabar as 5 tentativas

## Tecnologias

- Java
- Scanner
- Random

## Como executar

```bash
javac JogoAdivinhacao.java
java JogoAdivinhacao
